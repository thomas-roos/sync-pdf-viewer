PDF Sync for GitHub Pages

How it works:

  1. Presenter: Select section (1-14) and page (1-20), then generate QR code
  2. Viewers: Scan QR code or visit the URL on their devices
  3. All devices: Automatically load the corresponding PDF (e.g., 3_15.pdf) from their local folders

PDF Format: Files should be named like 1_1.pdf, 3_15.pdf, 14_20.pdf, etc.

⌨️ Keyboard Shortcuts (Presenter)
  • → or N - Next PDF
  • ← or P - Previous PDF
  • Home - Jump to start (1_1)
  • End - Jump to end (14_20)
  • Ctrl+Enter - Generate QR code
