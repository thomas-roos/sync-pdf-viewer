How it works:

  1. Presenter: Select section (1-14) and page (1-20), generate QR code
  2. Viewers: Scan QR code once to connect and load current PDF
  3. Real-time sync: All future navigation syncs instantly across all devices
  4. All devices: Load corresponding PDF (e.g., 3_15.pdf) from their local folders

PDF Format: Files should be named like 1_1.pdf, 3_15.pdf, 14_20.pdf, etc.
